## Overview

List is a view that displays a list (duh...) of items with some bells and whistles

TODO better overview screenshot
<UiScreenshot imgPath="/img/screenshots/docs_list/main/default.png"/>

### Functionality
- Items can have icons on left, title, subtitle and icons with optional text on right
- Items can be grouped in sections with titles and subtitles
- Can optionally have a search bar
- Can have a Detail side view that can change depending on focused item
- If no items exist, text with image can be shown in the middle

### Plugin Manifest

To use this view, entrypoint with type `"view"` is required. For more details on entrypoint see ... TODO

#### Example
```toml
[[entrypoint]]
id = 'main'
name = 'Main'
path = 'src/main.tsx'
type = 'view'
description = ''
```
